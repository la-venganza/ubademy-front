# ubademy-front
Taller 2 - 2c 2021 - Frontend repository

## Requirement

- Node v14 and up (you might want to take a look at https://github.com/nvm-sh/nvm)
- NPM

## Installation

- Clone the project
- Run `npm install`
- Run `npm run start` for development purposes. it should launch a server at 8080.
